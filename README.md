# sorteador-numeros

Objetivo:
- Um site que gera uma determinada quantidade de números aleatorios.

Como Funciona:
- Você determina a quantidade de números diferentes à serem gerados.
- Escolhe entre quais números eles serão gerados.
- O site informa os números gerados.

O que foi usado:
- JavaScript
- HTML
- CSS

Como Iniciar:
1. Baixe a pasta de arquivos.
2. Execute o arquivo index.html no navegador.

Melhorias Futuras:
- Aprender a usar o CSS e o HTML para editar o fundo, podendo deixar até uma interface de roleta girando ou algo do tipo para ficar mais interativo.
- Adicionar mais funcionalidades, exemplo: Somente gerar números pares ou impares
